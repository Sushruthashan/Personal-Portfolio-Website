Projects

Image Forgery Detection using CNN

This project focuses on developing a system to detect tampered or forged regions in digital images using deep learning techniques. By leveraging Convolutional Neural Networks (CNNs) for feature extraction , the model can identify common forgery techniques like copy-move, splicing, and image retouching. Using publicly available datasets like CASIA or CoMoFoD, the system generates a heatmap highlighting manipulated areas, aiding in the verification of image authenticity. This solution is valuable for journalism, law enforcement, and forensic investigations to combat misinformation and identify altered evidence.

Vulnerability Scanner

This Website Vulnerability Scanner is designed to identify security vulnerabilities in web applications, servers, and websites, helping developers and security professionals secure their systems from potential attacks. It performs automated scans for common vulnerabilities like SQL injection, XSS, and CSRF, detects misconfigurations, and uses tools like Nmap, SQLmap, and OWASP ZAP for deeper analysis. The scanner provides real-time reports with severity ratings and remediation suggestions, allowing security teams to address risks promptly. It supports both web-based and command-line interfaces, enhancing security by ensuring compliance with standards like OWASP Top 10 and PCI DSS.
